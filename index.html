This website was created for the mobile game Digimon Links to ease the viewing of digivolution lines and to assist with the awakening process.

Data and assets are intermittently gathered from Growlmon.Net, so this website may contain slightly outdated information.
The last database update was on a Tuesday.

To see the latest updates or send a bug report, visit this project's GitHub repository.

You can also send me a message through krazete@gmail.com, Reddit, or PayPal.
Q&A

Q: What is that circled exclamation mark in the corner of some Digimon cards?
A: This indicates the Digimon is currently available in Advent Quests (or will be within 12 hours).

Q: What does the Tree option in the Special search filter do?
A: Enabled this if you want the search results to display only Digimon of your currently selected digivolution tree.

Q: Isn't the Untangled (Beta) sort basis supposed to untangle the lines? It doesn't work too well.
A: Sorry, I don't know how to improve it. A greedy algorithm is the best I can do for now.

Q: What does the Preview option do?
A: With this enabled, hovering your cursor over a Digimon card will show that single Digimon's digivolution tree within the context of your currently selected digivolution tree.

Q: Why can't I use the Preview feature in search mode?
A: For clarity. Viewing digivolution lines in search mode would present an incomplete and misleading tree.

Q: I found a bug. Where can I report it?
A: Please send a report through krazete@gmail.com, Reddit, or GitHub and be sure to mention the browser in which you experienced the bug.
