This website was created for the mobile game Digimon Links to ease the viewing of digivolution lines and assist with the awakening process.
Data and assets are intermittently gathered from Growlmon.Net, so this website may contain slightly outdated information.
The last database update was on a Tuesday.

For more information on the latest updates, view this project's GitHub repository.

Q&A

Q: What is that circled exclamation mark in the corner of some Digimon cards?
A: This indicates that the Digimon is currently available in Advent Quests (or will be within 12 hours).

Q: What does the Tree option in the Special search filter do?
A: It narrows down your search to just the Digimon of the currently selected digivolution tree.

Q: Isn't the Untangle (Beta) sort basis supposed to untangle the lines? It doesn't work too well.
A: Sorry, I can't figure out how to make it any better. A greedy algorithm is the best I can do for now.

Q: What does the Preview feature do?
A: With Preview on, hovering your mouse over a Digimon will show that single Digimon's digivolution tree within the context of the currently selected digivolution tree.

Q: Why can't I use the Preview feature in search mode?
A: For clarity. Viewing digivolution lines in search mode would present an incomplete and misleading tree.

Q: Why does this website look so janky?
A: I only tested this in Chrome.
