This website was created by Krazete for the mobile game Digimon Links to ease the viewing of digivolution lines and to assist with the awakening process.
Data and assets are provided and periodically updated by Chortos-2.

To see the latest updates or send a bug report, visit this project's GitHub repository.
You can also send me a message through krazete@gmail.com, Discord @#4768, Reddit, or PayPal.
Found a bug? Report it through krazete@gmail.com, Reddit, or GitHub and mention which browser you're using.

Q: What's that number in the top right corner of some Digimon cards?
A: That's an Advent Schedule countdown. Blue indicates the Days, Hours, or Minutes until that Digimon's advent quest begins; orange shows when a current advent will end.

Q: Why are some Digimon cards semitransparent?
A: Those are Digimon that haven't yet been released in the global version of Digimon Links.

Q: What do the Personal search filter options do?
A: Select Tree to limit results to your currently selected digivolution tree. Select Fragments to hide Digimon who require fragments of which you have none of.

Q: Why doesn't the Untangled (Beta) sort option untangle the lines fully?
A: Sorry, I don't know how to improve it. A greedy algorithm is the best I can do for now.

Q: Why can't I use the Connections+ feature in search mode?
A: For clarity. Viewing digivolution routes in search mode would present an incomplete and misleading tree.
