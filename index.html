This website was created for the mobile game Digimon Links to ease the viewing of digivolution lines and to assist with the awakening process.
Data and assets are automatically gathered every month thanks to Chortos-2, so this website is always up-to-date.

To see the latest updates or send a bug report, visit this project's GitHub repository.
You can also send me a message through krazete@gmail.com, Reddit, or PayPal.
Found a bug? Report it through krazete@gmail.com, Reddit, or GitHub and mention which browser you're using.

Q: What's that circled "!" in the corner of some Digimon cards?
A: That Digimon is currently available in Advent Quests (or will be within 12 hours).

Q: What does the Tree option in the Special search filter do?
A: Enable this if you want the search results to display only Digimon of your currently selected digivolution tree.

Q: Why doesn't the Untangled (Beta) sort option untangle the lines fully?
A: Sorry, I don't know how to improve it. A greedy algorithm is the best I can do for now.

Q: Why can't I use the Preview feature in search mode?
A: For clarity. Viewing digivolution lines in search mode would present an incomplete and misleading tree.

Q: Why aren't my settings/planner/fragments being automatically saved?
A: This is a known bug. Autosaving happens when the page is closed, which is difficult to detect on mobile devices.
